PROYECTO COMPILADOR

APACHE JAVA REPOSITORIO

Detalles del Proyectos

Este proyecto se enfoca en el desarrollo de un analizador que abarca tres niveles esenciales en el procesamiento de código: análisis léxico, sintáctico y semántico. Su función principal es recibir como entrada código en lenguaje ensamblador, para luego analizar y traducirlo. En la etapa léxica, el programa identifica y clasifica cada elemento del código, como palabras clave, operadores y delimitadores. Durante el análisis sintáctico, se verifica que la estructura del código cumpla con las reglas gramaticales propias del lenguaje ensamblador. Finalmente, en la fase semántica, se valida que el significado del código sea coherente y correcto, asegurando que las instrucciones ensambladas sean ejecutables de manera precisa. Este proceso integral permite una traducción fiel y precisa del código, facilitando así su comprensión y uso en otros contextos.
