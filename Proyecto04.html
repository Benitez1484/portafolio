PROYECTO ABARROTERIA

VISUAL STUDIO C++ REPOSITORIO

Detalles del Proyectos

Este sistema permite gestionar datos en la "Tienda de Abarrotería Don Cabal" mediante un menú interactivo basado en el método de pila. A través de este menú, el usuario puede ingresar, visualizar, modificar, buscar y eliminar datos de manera secuencial, siguiendo la lógica de último en entrar, primero en salir (LIFO). Además, ofrece una opción para regresar al menú principal. Este sistema facilita una gestión simple y estructurada de la información en la tienda, permitiendo realizar operaciones básicas de manera intuitiva y organizada.
